# 🌸 Iris Flower Classification using Machine Learning

## Oasis Infobyte Data Science Internship - Task 1

---

# 📌 Project Overview

This project builds a Machine Learning model to classify Iris flowers into three species based on their sepal and petal measurements. The Iris dataset is one of the most popular datasets used for learning classification algorithms.

The objective is to perform data analysis, visualize the dataset, train a classification model, and accurately predict the species of Iris flowers.

---

# 🎯 Objectives

- Analyze the Iris dataset.
- Perform Exploratory Data Analysis (EDA).
- Visualize feature relationships.
- Train a Machine Learning classification model.
- Predict Iris flower species.
- Evaluate the model's performance.

---

# 📂 Dataset

The project uses the famous **Iris Dataset** containing 150 flower samples.

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Classes

- Iris-setosa
- Iris-versicolor
- Iris-virginica
